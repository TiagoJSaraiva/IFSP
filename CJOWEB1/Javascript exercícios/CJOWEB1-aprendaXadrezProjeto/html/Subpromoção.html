Subpromoção
Promover (veja "promoção") um peão para uma peça que seja menor que uma Dama (por outras palavras, promover um peão para um Cavalo, Torre, ou Bispo). Uma vez que a Dama é a peça mais forte, os jogadores quase sempre escolhem promover os seus peões para Damas. Uma tática de subpromoção ocorre quando existem razões especiais em que um jogador precisa duma peça mais fraca do que uma Dama (quase sempre um Cavalo, uma vez que é a única peça cujo movimento não está já refletido nas capacidades de uma Dama) - para impedir a ameaça dum oponente ou para alcançar algo ainda melhor daquilo que uma Dama poderia oferecer.
A ideia chave deste tema tático: as pretas "subpromovem" para um Bispo com um objetivo especifico, que é de impedir o truque de afogamento das brancas.
