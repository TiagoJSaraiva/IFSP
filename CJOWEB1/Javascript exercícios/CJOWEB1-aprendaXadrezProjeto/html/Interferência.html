Interferência
Mover uma peça entre duas outras peças atacantes quando pelo menos uma destas peças atacantes é uma peça do oponente. Por vezes uma peça irá interferir com duas peças atacantes, criando portanto confusão e sobrecarregando com frequência (ver "sobrecarregar") aquelas peças.
