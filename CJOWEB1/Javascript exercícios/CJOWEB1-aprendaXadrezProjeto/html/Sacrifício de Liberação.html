Sacrifício de Liberação
"Sacrifício de Liberação" (ou simplesmente "liberação") é um termo usado para descrever um sacrifício deliberado de material com o objetivo de "abrir" uma casa, uma diagonal, ou uma coluna. Os sacrifícios para abrir espaço mais comuns abrem uma diagonal crítica (ver exemplo). O sacrifício de um peão para abrir uma casa para um Cavalo (tal como um peão que se move para e5 a partir de e4, sacrificando-se de modo a libertar a casa-e4 para um Cavalo) também seria considerado como um sacrifício "posicional" para abrir espaço.
A ideia chave deste tema tático: as brancas "abrem" a longa diagonal a2-g8 ao sacrificar a Torre em f8. Agora as brancas terminam a partida com um mate de cavalo com o rei preso no canto.
