Notação algébrica
A notação algébrica é o sistema padrão para registrar os movimentos das peças no xadrez, e é usada universalmente em partidas, livros, e programas de xadrez. Essa notação ajuda a descrever qualquer movimento de maneira simples e precisa. Vamos ver como ela funciona:

O Tabuleiro e as Coordenadas

- O tabuleiro é dividido em uma grade de 8x8.
- As colunas são identificadas com letras de a a h (da esquerda para a direita).
- As linhas são numeradas de 1 a 8 (de baixo para cima, do ponto de vista das brancas).
- Cada casa, então, tem uma "coordenada" única. Por exemplo, a casa no canto inferior esquerdo é a1, enquanto a casa no canto superior direito é h8.
Movimentos das Peças

Cada peça é identificada por uma letra maiúscula:

K = Rei (do inglês King)
Q = Dama (Queen)
R = Torre (Rook)
B = Bispo (Bishop)
N = Cavalo (Knight; cavalo é "knight" em inglês, e "K" é reservado para o rei)
P = Peão (geralmente não se usa uma letra para peões; apenas o movimento é indicado)

Para registrar o movimento de uma peça, é escrita a letra da peça seguida da casa de destino. Por exemplo:

- Qd4 significa que a dama se move para a casa d4.
- Nf3 significa que o cavalo se move para f3.
- e4 (sem letra) indica que um peão move-se para a casa e4
Capturas

Uma captura é indicada pela letra da peça seguida de um "x" e a casa onde ocorreu a captura.
Exemplo: Bxe5 significa que o bispo capturou uma peça na casa e5.
Para peões, o movimento inclui a coluna de origem. Por exemplo, exd5 significa que o peão da coluna e capturou uma peça em d5.
Movimentos Especiais

Roque: Existem dois tipos de roque.

Roque menor (curto) é anotado como O-O.
Roque maior (longo) é anotado como O-O-O.

Promoção de Peão: Quando um peão chega à oitava linha, ele é promovido, geralmente para uma dama. A promoção é indicada pelo movimento do peão seguido da peça escolhida, por exemplo, e8=Q.

Xeque e Xeque-mate:

Um xeque é indicado com um sinal de +. Exemplo: Qe5+ (a dama se move para e5 e coloca o rei em xeque).
Xeque-mate é indicado com #, como Qh5#.
